<h1 style="font-size: 36px;">Personal Portfolio Website - Test Automation Engineer</h1>

<h2 style="font-size: 24px;">Features</h2>

<h3 style="font-size: 18px;">Home Page</h3>

<p>Provides an overview of the engineer's skills, experience, and achievements. It also displays links to the engineer's social media profiles.</p>

<h3 style="font-size: 18px;">About Page</h3>

<p>Provides more information about the engineer's professional background, skills, and interests.</p>

<h3 style="font-size: 18px;">Projects Page</h3>

<p>Showcases the engineer's notable projects related to test automation. Each project provides a brief description, screenshots, and links to the GitHub repository and the live demo (if applicable).</p>

<h3 style="font-size: 18px;">Resume Page</h3>

<p>Provides a downloadable copy of the engineer's resume.</p>

<h3 style="font-size: 18px;">Contact Page</h3>

<p>Provides a form for visitors to send a message to the engineer.</p>

<h2 style="font-size: 24px;">Technologies Used</h2>

<p>The website is built using the following technologies:</p>

<ul>
  <li>HTML5 and CSS3 for the website layout and styling.</li>
  <li>JavaScript for the interactive elements, such as the navigation menu, scroll-to-top button, and form validation.</li>
  <li>Bootstrap framework for the responsive design and layout.</li>
</ul>

<h2 style="font-size: 24px;">Deployment</h2>

<p>The website is deployed using GitHub Pages. The source code is hosted on GitHub, and the website is automatically built and deployed whenever changes are pushed to the main branch.</p>

<h2 style="font-size: 24px;">Credits</h2>

<p>The website design and layout were inspired by various portfolio templates and themes available online. The project screenshots were taken from the engineer's actual projects.</p>

<h2 style="font-size: 24px;">Conclusion</h2>

<p>This personal portfolio website serves as a showcase of the engineer's skills, experience, and achievements in the field of test automation. It provides an easy-to-navigate and visually appealing platform for potential employers and collaborators to learn more about the engineer and their work.</p>
